🎬 Hexlet Series
Веб-приложение для отслеживания просмотренных сериалов

Hexlet Series — это учебное веб-приложение, разработанное в рамках курсовой работы.
Сайт создан с использованием HTML5, CSS3 и Vanilla JavaScript и предназначен для добавления сериалов, отслеживания просмотренных серий и сохранения прогресса в браузере.

🚀 Функционал проекта
✔️ Главная страница
информационный блок о приложении;
описание возможностей сайта;
простая и понятная навигация;
адаптивный дизайн.
✔️ Каталог сериалов
форма для добавления нового сериала;
ввод названия сериала;
ввод общего количества серий;
ввод количества просмотренных серий;
добавление URL изображения сериала;
автоматическое создание карточки через JavaScript.
✔️ Работа с карточками
отображение названия сериала;
отображение изображения;
вывод количества просмотренных и общих серий;
автоматический расчёт прогресса в процентах;
кнопка +1 серия;
удаление сериала из списка.
✔️ Завершение просмотра
если количество просмотренных серий равно общему количеству серий, карточка автоматически получает зелёный фон;
появляется надпись «Сериал завершён ✔»;
кнопка добавления серии становится неактивной.
✔️ Работа с данными
сохранение сериалов в localStorage;
данные не теряются после перезагрузки страницы;
возможность хранить собственную коллекцию сериалов в браузере.
✔️ Мобильное меню (бургер)
открытие и закрытие меню;
анимация бургер-кнопки;
адаптация под мобильные устройства;
блокировка прокрутки страницы при открытом меню.
✔️ Дополнительные страницы
Главная — основная информация о проекте;
Каталог — добавление и просмотр сериалов;
О проекте — описание назначения сайта;
Контакты — форма обратной связи.
✔️ Модальное окно
кнопка «Профиль» открывает модальное окно;
форма входа и регистрации;
закрытие по кнопке и по клику вне окна.
📱 Адаптивность
Сайт корректно отображается на:

📱 мобильных телефонах;
📲 планшетах;
💻 ноутбуках;
🖥️ больших экранах.
Используются:

Flexbox;
Grid;
Media queries.
🛠️ Технологии
Технология	Использование
HTML5	структура страниц
CSS3	стилизация, адаптивность, сетки
JavaScript	динамическое добавление сериалов, работа с карточками, бургер-меню
localStorage	сохранение данных пользователя в браузере
